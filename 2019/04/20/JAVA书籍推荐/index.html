JAVA书籍推荐

2019-04-20 图书推荐

一、《Head First Java》中文版

无论你之前是否接触过任何编程语言，这本书都适合初学者。

通过这本书你可以在轻松、愉快的氛围中学会Java基础语法，并首次体验Java面向对象编程思想。

二、《Head First 设计模式》

通过阅读上一本的《Head First Java》，可以使我们对Java编程有了初步了解，我们要知道，书是拿来读的，如果一上来就把读者吓住了，难啃难咽，这肯定不是读者喜欢的，与上本书同系列的《Head First 设计模式》通过简单有趣的方式向我们阐述了Java编程的关键知识，有助于我们进一步对Java编程的认知，嘿嘿，通过打怪我们又升了一级。

三、《Java编程思想》

其实关于这本书，原名为《thinking in Java》，别像小编当初一样，只知道《Java编程思想》，到了书店手里拿着《Java编程思想》问老板有没有《thinking in Java》，呵呵，其实这本书网上有争议，认为这本书不应该推荐初学者阅读，亦是美网络小编不这样认为，因为这本书可以贯穿你整个学习Java编程的过程，即使你进阶成为了高级程序员，这本书既可以用来作为学习java的教程，又可以作为工具书进行知识点查询，当然了，如果亲们的英语能力还可以，强烈建议读英文原版咯。

四、《疯狂Java讲义》

对于这本书来说，怎么评价呢，说不上来的感觉，但它充斥了全国各大书店计算机区域，自从推出就一直热度不错，读这本书更像是接受培训一样，不过这样也好，通过上面几本书的入门和认知，再被这本书培训一下，有什么不好吗？

五、《Java深度历险》

这本书听名字就感觉很高深的样子，学习个Java编程都要历险了，这本书写的也很不错，但不要用于基础学习，可以当做业务读物没事翻翻看看，或许会给你一些臆想不到的灵感。
